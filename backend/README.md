# Code Galaxy API

Backend engine for the Code Galaxy 3D visualization suite.

## Tech Stack
- **Framework**: FastAPI
- **Intelligence**: Groq Llama 3.1
- **Database**: SQLite (aiosqlite)
- **Geometry**: UMAP + scikit-learn

## Deployment
This branch is automatically deployed to Hugging Face Spaces via Docker.
